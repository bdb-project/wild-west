Wild West, release 2
By Gianluca Girelli & Garry Francis
For Z-code interpreters

Solution by Garry Francis
----------------------------------------
This is a minimal solution. You'll need to EXAMINE lots of things for this solution to make sense. Some optional (but significant) commands are shown in parentheses. These can be omitted once you know what to do.

The game starts off in the desert. You can't map it in the conventional manner by dropping objects to make each room unique because you don't have many items. However, if you examine the desert, you'll find some hoof prints, so follow them. You can return later, when you have more objects, to map it properly and find a slightly shorter route.

NOTE: There are some small changes compared to release 1, particularly in the stagecoach and the end game. There's also a more extensive HINT system.
